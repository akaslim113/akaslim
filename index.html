소개

이 사이트는 무엇인가요?

안녕하세요. 저는 개발을 전문으로 배우지 않은 평범한 사용자예요. 그런데도 AI 코딩 툴인 클로드(Claude)와 대화하면서, 실제 음식점에서 쓰는 예약 앱을 만들고, 유튜브 쇼츠까지 자동으로 찍어내고 있어요.

이 사이트는 그 과정을 솔직하게 기록한 공간이에요. 막혔던 부분, AI에게 어떻게 부탁했는지, 무엇을 배웠는지를 초보자의 눈높이에서 풀어 씁니다. 저처럼 '코딩은 모르지만 만들고 싶은 게 있는' 분들에게 도움이 되면 좋겠어요.

지금은 크게 두 갈래의 작업을 하고 있어요. 하나는 식당에서 쓰는 예약 앱, 다른 하나는 클로드로 돌리는 콘텐츠 자동화예요. 위 메뉴에서 갈래별로 골라 볼 수 있어요.

이 블로그도 클로드와 이렇게 만들었어요

거창한 코딩이 아니라, 그냥 말로 부탁하고 결과를 보고 또 고쳐달라고 했어요. 실제로 오간 대화 일부예요.

나 홈페이지 하나 만들어서, 내가 작업한 내용들을 블로그 글처럼 자동으로 쓰게 해줘.
클로드 작업 기록(git 커밋)을 읽어서, 초보자가 AI로 앱 만드는 과정을 정적 HTML 블로그로 자동 생성하는 스크립트를 만들었어요. 6일치 개발일지가 나왔어요.
나 근데 홈페이지 같지가 않아서 좀 밋밋해. 더 그럴듯하게, 실제로 접속해서 볼 수 있게 해줘.
클로드 상단 메뉴·히어로·소개·프로젝트 구조로 다시 디자인하고, GitHub Pages로 인터넷에 공개하는 것까지 준비했어요.
나 글이 너무 AI처럼 쓰여서 좀 더 사람처럼, 사진 같은 것도 넣어줘.
클로드 여섯 개 글을 구어체로 다시 쓰고, 쇼츠가 실제로 만든 이미지랑 화면 사진 자리를 곳곳에 넣었어요.
나 작업한 거 매일 오전 9시에 자동으로 올라가게 해줘.
클로드 매일 9시에 사이트를 새로 만들어 GitHub에 자동으로 올리도록 맥에 예약을 걸었어요.
